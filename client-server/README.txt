1. Запустить из папки server приложение spring-server-0.0.1-SNAPSHOT.jar
2. Перейти по адресу: localhost:8080/summary/фамилия/имя/отчество, подставить ФИО. 
3. Проверить добавленные данные можно по адресу: localhost:8080/info
4. Запустить клиент, зайти по адресу localhost:4200, нажать кнопку "Получить данные с сервера"
